# function-hcl

<!-- Plugin description -->
IntelliJ plugin for the function-hcl DSL. Provides language support via the
[function-hcl-ls](https://github.com/crossplane-contrib/function-hcl) language server.

Features:

- HCL syntax highlighting (via Terraform plugin)
- Code completion
- Hover information
- Go to definition
- Find references
- Document symbols
- Diagnostics
- Code formatting
<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "function-hcl"</kbd> >
  <kbd>Install</kbd>

- Manually:

  Download the [latest release](https://github.com/crossplane-contrib/function-hcl/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
